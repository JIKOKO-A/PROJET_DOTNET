## ğŸ“ Summary

Each page follows a similar pattern:
1. **Initialization** - View created, ViewModel instantiated, database context created
2. **Data Loading** - Queries database, loads into ObservableCollections
3. **User Interaction** - User clicks buttons, selects items, fills forms
4. **Validation** - Checks required fields, duplicates, business rules
5. **Database Operations** - Add/Update/Delete using Entity Framework
6. **Refresh** - Reloads data, updates UI automatically
7. **Feedback** - Shows success/error messages

All file paths are relative to the project root: `HRManagementApp/`
